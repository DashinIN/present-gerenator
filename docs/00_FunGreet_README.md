## Ключевые технические решения

1. **Монолит на старте, не микросервисы.** Одна Go-бинарка с HTTP + Worker.

2. **gcui-art/suno-api для музыки.** Self-hosted прокси снижает стоимость песни с $0.11 до $0.02. Trade-off: нужно следить за cookie.

3. **Nano Banana через free tier.** До 500 генераций картинок в день бесплатно. Переход на $0.039/img по мере роста.

4. **httpOnly cookies для токенов.** Безопаснее localStorage от XSS. Требует настройки `SameSite=None; Secure; Partitioned` для совместимости с TG Mini App.

5. **Отделение users от user_identities.** Один пользователь = одна запись, к которой привязаны N способов входа (Google + VK + Telegram).

6. **Redis для очередей и rate limiting.** Простое решение для масштаба MVP.

7. **PostgreSQL как source of truth.** Все критичные данные здесь, R2 — только файлы, Redis — эфемерное.

8. **Docker Compose + Traefik на одном VPS.** Оверкилл в виде Kubernetes не нужен до 10K пользователей/день.

## Критические зависимости

| Зависимость | Риск | План B |
|-------------|------|--------|
| **Suno (через cookie)** | Высокий — Suno может закрыть обход | Переход на EvoLink/Apiframe ($0.11/песня), рост себестоимости |
| **Gemini API** | Низкий | Imagen 4 Fast ($0.02) или fal.ai SDXL |
| **Google OAuth** | Низкий | VK + Yandex покрывают русскую аудиторию |
| **Cloudflare R2** | Низкий | Любой S3-compatible (Backblaze B2, Wasabi) |
